## Bugs Fixed

### MQTT Auth
- **File**: `backend/app/core/mqtt.py`
- **Issue**: `client.connect(username=..., password=...)` is unsupported in gmqtt
- **Fix**: Call `client.set_auth_credentials(username, password)` BEFORE `connect()`
- **Scope**: Both initial connect and reconnect loop

### crop_type ENUM mismatch
- **File**: `backend/app/domain/fields/models.py`
- **Issue**: PG column created as ENUM via Alembic, SQLAlchemy model used `String(50)`
- **Fix**: Changed to `Enum("banana","maize","cacao","rice", name="crop_type", create_type=False)`

### signal_quality CAST syntax
- **File**: `backend/app/domain/ingestion/service.py`
- **Issue**: `:signal_quality_i::jsonb` mixed named params with PostgreSQL `::` cast
- **Fix**: `CAST(:signal_quality_i AS jsonb)`

### Global Exception Handler
- **File**: `backend/app/main.py`
- **Fix**: Added `@app.exception_handler(Exception)` with logger.exception() + JSON response
